# neuro

Deep Neural Network written in Clojure from scratch

[![Clojars Project](https://img.shields.io/clojars/v/deltam/neuro.svg)](https://clojars.org/deltam/neuro)

## やったこと

* mnist を読み込む
* 勾配降下法で二値分類を学習
* 確率的勾配降下法で学習する
* 誤差逆伝播法で学習する
* 多クラス分類
* mnistから学習して手書き文字認識をする

## これからやりたいこと

* ONNXモデルを読み込む
